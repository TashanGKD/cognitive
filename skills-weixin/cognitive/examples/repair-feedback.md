# Example: Repair feedback

Owner feedback:

> 这不像我，太像客服了，而且太热情。我一般不会说那么多鼓励的话。

Expected behavior:

1. Diagnose the failure as expression DNA plus relationship-distance mismatch.
2. Produce a corrected answer immediately if possible.
3. Ask for one real counterexample only if more evidence is needed.
4. Persist the repair only after owner authorization.

Possible repair note:

> 这次主要错在距离和密度：用了过多安抚句，判断也不够直接。我会把默认风格调成更短、更克制、先判断再解释，并把“空泛鼓励”加入 anti_patterns。
